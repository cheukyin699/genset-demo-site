Instructions

Basic Usage

Once you have the testing set and the training set (per specifications), you may start uploading.

  1. Go to Upload tab
  2. Upload testing set CSV
  3. Select organism to train against OR select Custom and upload your custom training set
  4. Press Upload button
  5. Wait until progress bar goes to 100%
  6. Click Download button
  7. After successfully saving results, click Delete button

All uploaded files will be deleted automatically 1 (one) day proceeding the upload, regardless of whether the run was successful or not.

Advanced Usage - More than 20 attributes

For the slightly advanced user, you may think that 20 attributes are far from enough. In this case, there are a few things that you'd ought to know.

When CSV files are uploaded, they are sent to a directory and preprocessed. A script removes the protein name (1st column), temporarily. The effector attribute is assumed to be the last column.

In order to add a new attribute, you must insert columns between the 14th column and the last column.

In order to remove an existing attribute, you must replace said column with a column full of useless values. If the column you need to remove is to the left of the 14th column you can remove it just fine. If it is to the right of the 14th column, you can switch said column with a needed column to the right of the 14th column, and delete the column.
