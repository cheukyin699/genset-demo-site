About the Project

By using family specific prediction methods, we are able to improve effector predictions with other published programs. This approach can be applied to predict effectors of other secretion systems as long as these effectors have translocation signals embedded in their sequences.

This site was created in order to make the entire WEKA training/predicting process more streamlined. 3 processes are automated: a) file type conversion, b) the WEKA run itself, and c) result reformatting. These 3 once tedious processes are now completely automatic.

There are essentially 2 steps to the GenSET method: attribute collection and machine learning. This site only covers the machine learning part; attribute gathering must be done separately, as per the instructions and specifications here.
